   ```bash
   git clone https://github.com/krishnavamsi333/Portfolio.git

   ```

   ```bash

   ```

   ```bash
   npx http-server
   ```

   ```bash
   php -S localhost:8000
   ```

## ğŸ¨ Customization

### Updating Content

1. **Personal Information**: Edit `components/header.html` and `components/about.html`
2. **Skills**: Modify `components/skills.html`
3. **Projects**: Update `components/projects.html`
4. **Publications**: Edit `components/publications.html`
5. **Contact**: Modify `components/contact.html`

### Styling

- **Colors**: Edit CSS variables in `assets/css/base.css`
- **Fonts**: Change font imports in `index.html`
- **Animations**: Modify `assets/css/animations.css`
